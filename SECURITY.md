# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please report it privately.

**Do not report security vulnerabilities through public GitHub issues.**

Instead, send an email to **hunnu-lang@proton.me** with:

- A description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any suggested fix (if known)

You will receive a response within 48 hours. We will keep you informed of the progress towards a fix and release.

## Preferred Languages

English or Mongolian.
